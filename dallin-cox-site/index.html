Healthcare Operations

Prior Authorization Tracking — Automating a manual high-volume workflow.

Process mapping · System architecture · Drug-payer matrix

A full process redesign for prior authorization tracking at a high-volume specialty practice processing 30,000+ annual patient visits. Mapped the existing manual spreadsheet workflow — color-coded by hand, sorted and re-sorted daily, with all expiration logic living in one staff member's head — then architected a future-state automation that keeps human judgment where it matters (clinical review, submission, appeals) and removes it where it doesn't (tier calculation, ingestion, metrics).

  • 9-step manual workflow mapped end-to-end
  • Drug-payer reference matrix designed as canonical rule source
  • Four-tier urgency system: immediate / urgent / warning / monitored
  • Per-eye PA data model (OD / OS / OU laterality tracking)
  • Excel-based operational store sized for 30k+ annual visits
  • Automated tier calculation: PA expiry vs. next appointment date
  • Denial / appeal / peer-to-peer branch architected into workflow
  • Metrics writeback: turnaround time, approval rate, denial rate
  • EHR-agnostic ingest (works with any tabular export)
  • Manual workflow documented for stakeholder validation
